Basic operations
Insert or delete bar

  • Insert bar
  • Deleta bar

Insert bar

1. Go to the select mode().

2. Right-click where you want to be insert new bar, and select "Insert part". (the bar where right-clicked would be shifted to upper one)

Delete bar

1. Go to the select mode ().

2. Right-click where you want to be insert new bar, and select "Insert part". (the bar where right-clicked would be shifted to upper one)

Remarks

  • If you delete the bar which has some chips, the chips would be deleted, too.
